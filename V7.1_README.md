# Dofus Craft Manager V7.1

## Nouveautés

- identité locale demandée une seule fois par appareil ;
- affichage des utilisateurs connectés en haut à droite ;
- présence actualisée automatiquement toutes les 15 secondes ;
- activité quotidienne regroupée par utilisateur ;
- dernières modifications affichées dans le grand bloc du Dashboard ;
- statistiques de fraîcheur : moins de 24 h, de 1 à 7 jours, plus de 7 jours ;
- historique des prix enrichi avec le nom de l'utilisateur ;
- migration automatique de la base V7.0 vers V7.1.

Le nom choisi est enregistré dans le stockage local du navigateur. Aucun mot de passe n'est requis dans cette version destinée à un groupe privé de confiance.
